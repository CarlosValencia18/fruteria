Description
Main DescriptionEl Diagrama de Casos de Uso es una herramienta de modelado que muestra las interacciones entre los actores (usuarios y sistemas externos) y el sistema en el contexto del proceso de compra de insumos a proveedores en la frutería. Utiliza casos de uso para representar las diferentes funciones y acciones que los usuarios pueden llevar a cabo con el sistema, ayudando a definir los requisitos funcionales del sistema desde una perspectiva de usuario. Los actores representan roles o entidades externas que interactúan con el sistema, mientras que los casos de uso representan las funciones o acciones que el sistema debe realizar para satisfacer las necesidades del usuario. El Diagrama de Casos de Uso proporciona una visión general de alto nivel del sistema y ayuda a identificar los requisitos funcionales clave que deben ser implementados
