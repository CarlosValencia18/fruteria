Artifact: Prototipo UI
El Prototipo de Interfaz de Usuario (UI) es una representación visual de la interfaz de usuario del sistema utilizado durante el proceso de compra de insumos a proveedores en la frutería
Purpose
Su propósito es proporcionar una vista preliminar de cómo será la interfaz de usuario final, permitiendo a los usuarios y diseñadores visualizar y probar la funcionalidad y el flujo de la aplicación

Description
Main DescriptionEl Prototipo de Interfaz de Usuario (UI) es una maqueta visual o interactiva que muestra cómo será la interfaz de usuario del sistema durante el proceso de compra de insumos a proveedores en la frutería. Puede incluir elementos como pantallas, botones, campos de entrada y otros componentes de la interfaz, diseñados para representar la apariencia y la interacción del sistema final. El prototipo puede ser de baja fidelidad (bocetos simples) o de alta fidelidad (diseños más detallados y realistas), y puede ser estático o interactivo, dependiendo de las necesidades del proyecto. El prototipo de UI es una herramienta invaluable para validar y refinar los requisitos del usuario, comunicar el diseño del sistema a los interesados y realizar pruebas de usabilidad antes de la implementación final
