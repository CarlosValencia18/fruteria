Artifact: Casos de uso de sistema
Los Casos de Uso del Sistema describen las interacciones entre los usuarios y el sistema durante el proceso de compra de insumos a proveedores en la frutería.
Purpose
Su propósito es detallar las acciones específicas que los usuarios pueden realizar y cómo el sistema responde a esas acciones, facilitando el diseño y desarrollo del sistema

Description
Main DescriptionLos Casos de Uso del Sistema son una técnica de modelado que describe las interacciones entre los usuarios y el sistema en el contexto del proceso de compra de insumos a proveedores en la frutería. Cada caso de uso representa una funcionalidad específica del sistema, mostrando cómo un usuario interactúa con el sistema para lograr un objetivo particular. Esto incluye las acciones que realiza el usuario, las respuestas del sistema y cualquier resultado esperado. Los casos de uso del sistema proporcionan una comprensión detallada de las funcionalidades del sistema desde la perspectiva del usuario, ayudando a definir los requisitos del sistema y a guiar el diseño, desarrollo y prueba del sistema de manera efectiva
